#####屏幕旋转 Activity 的生命周期

看生命周期可以发现，在 Activity 正常销毁时候不会掉用 onSaveInstanceState，所以我们可以了理解成在 Activity 异常销毁时才会执行 onSaveInstanceState 和 onRestoreInstanceState 来存储和恢复数据。

注：如果不想系统重新创建 Activity ，可以给 Activity 指定 configChanges 属性，指定多个熟悉可以用 | 拼接

  • 我们试试加上 configChanges ，根据书本中内容介绍的熟悉，样例编译的 API 都大于 13 ，所以我们得加上 screenSize
